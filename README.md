# Note & Photo Gallery App

Приложение с фотогалереей и возможностью хранить записи, построенное по принципам Clean Architecture. Переключение между разделами — свайпом. Все данные хранятся в приватной папке приложения.

##  Функционал

### Дневник
- **Список записей** — загружается при старте, показывает дату и превью текста
- **Создание записи** — FAB-кнопка, поле заголовка (опционально) + поле текста
- **Сохранение** — каждая запись в отдельном `.txt` файле с именем `timestamp_[заголовок].txt`
- **Просмотр и редактирование** — клик на запись открывает её содержимое
- **Удаление** — через контекстное меню по долгому нажатию

### Фотогалерея
- **Съёмка фото** — кнопка открывает камеру, снимок сохраняется автоматически
- **Сетка фото** — все снимки в сетке 3 колонки
- **Экспорт в галерею** — сохранение копии в стандартную галерею телефона
- **Разрешения** — приложение запрашивает `CAMERA`

## Хранение данных

| Данные | Где хранится |
|---|---|
| Записи дневника | `InternalStorage`, формат `timestamp_[заголовок].txt` |
| Фотографии | `ExternalStorage`, формат `IMG_yyyyMMdd_HHmmss.jpg` |
